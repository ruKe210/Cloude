# 第一幕「初来乍到」— 美术需求表

> **版本**：v1.2  
> **视觉基调**：明亮现代办公室、轻度像素/故障点缀  
> **原则**：第一天 **背景 + Meta 正面立绘 + 战斗序列帧**；**玩家/导师战斗只用序列帧，不要战斗侧面静态立绘**

---

## 1. 第一天交付目标

支撑 **Meta 入职段（S01~S02）+ 战斗1 导师带练** 跑通即可：

| 类别 | 第一天做 | 第一天不做 |
|------|----------|------------|
| 背景 | 2 张 Meta 背景 + 1 张战斗背景 | 会议室、机房、收工夜景 |
| 立绘 | 玩家、老陈 **Meta 正面**（仅对话） | 玩家/导师 **战斗侧面静态图** |
| 动画 | **玩家 + 导师 + 4 怪** 战斗序列帧 | UI 特效、卡牌面 |
| 敌人参考图 | 4 怪可选 **侧面设定稿**（进 `Sprites/` 首帧即可，不单独要 `Portraits/Battle/`） | — |

**资源目录**

```
Assets/Art/Act1/
├── Background/
├── Portraits/Meta/          # 仅 Meta 对话正面立绘
└── Sprites/                 # 战斗全部走序列帧
    ├── Player/              # 玩家：idle / attack / hurt(可选)
    ├── Mentor/              # 导师：idle / talk（指点提示，不参战无 attack）
    ├── Enemy_Error/
    ├── Enemy_404/
    ├── Enemy_Warn/
    └── Enemy_Spaghetti/
```

> **程序约定**：`WorldEntityView` / `SpriteSequenceAnimator` 用序列帧第一帧作初始姿态；**不再加载** `Portraits/Battle/player_side.png` 类静态图。

**序列帧规范**

| 项目 | 建议 |
|------|------|
| 单帧尺寸 | 256×256 或 512×512（玩家、Boss 可略大） |
| 最少动作 | 见下表各角色 |
| 格式 | PNG 透明底；横排图集或单动作单文件 |

| 角色 | 必备动作 | 说明 |
|------|----------|------|
| 玩家 | idle、attack | 战斗主控 |
| 导师 | idle、talk | 左侧场外；talk 在提示台词时播，**无 attack** |
| 怪物 | idle、attack | 与现设计一致 |

## 3. 制作顺序

```
背景 → Meta 正面
→ 玩家 + 导师序列帧（ART-D1-06、07）
→ 怪物序列帧 ERROR+404 → WARN → 屎山（ART-D1-11~14）
```

---

## 4. 程序对接

| 交付 | 程序侧 |
|------|--------|
| Meta 正面 | `NarrativeSceneController` 换立绘 |
| 玩家序列帧 | 战斗左侧实体，`WorldEntityView` 播 idle/attack |
| 导师序列帧 | 场外 `MentorBattleView`，默认 idle，台词事件播 talk |
| 怪物序列帧 | 敌人实体同上 |
