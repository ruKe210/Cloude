# 战斗原型 — 快速启动

## 1. 运行方式（零配置）

1. 打开 Unity 场景 `Assets/Scenes/SampleScene.unity`
2. 在 Hierarchy 创建空物体，命名为 `BattleRoot`
3. 挂载脚本 `STS.Presentation.BattleSceneController`
4. 保持 Inspector 中 **Auto Create Ui** 勾选
5. 点击 Play

进入 Play 后会自动创建 Canvas、手牌区、HUD，并开始一场 **Ironclad vs Jaw Worm** 战斗。

## 2. 操作说明

- 点击底部手牌出牌（攻击牌自动以 Jaw Worm 为目标）
- 点击「结束回合」进入敌人回合
- Console 会输出伤害日志

## 3. 已实现系统

- Action 队列 + cardQueue + UseCardAction
- 10 张初始牌组（5 Strike / 4 Defend / 1 Bash）
- 能量、抽牌、弃牌、洗牌
- 格挡、力量/虚弱/易伤
- Jaw Worm 条件 AI + Intent 显示
- 胜负判定

## 4. 代码目录

```
Assets/Scripts/
├── Core/           EntityId, BattlePhase, IBattleEntity
├── Data/           枚举定义
├── Entities/       Player, Enemy, CardPile
├── Powers/         Strength, Weak, Vulnerable
├── Battle/         BattleContext, BattleManager, CombatActionManager
├── Actions/        Damage, Block, UseCard, EnemyTurn...
├── Cards/          Strike, Defend, Bash
├── AI/             JawWormAI
└── Presentation/   场景入口与 UI
```
